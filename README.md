# Odin Recipes

This project is part of The Odin Project Foundations curriculum. The goal of this project is to build a basic recipe website using only HTML.

## Project Overview

The website will consist of a main index page that links to several recipe pages. Each recipe page will include headings, images, unordered and ordered lists, and links.

## Skills Demonstrated

By completing this project, I will have demonstrated:
- Basic HTML structure
- Proper use of HTML elements (headings, paragraphs, lists, links, images)
- File and directory organization
- Using Git for version control
- Using GitHub to host and manage a project repository
